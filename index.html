Introdução

Este módulo do curso é inspirado no programa Software Carpentry. A missão deste grupo é ajudar cientistas e engenheiros a se tornarem mais produtivos através do ensino de habilidades básicas de computação, como design de aplicativos, controle de versão, gerenciamento de dados e automatização de tarefas. Este curso prático cobrirá conceitos básicos e ferramentas; os alunos serão encorajados a ajudar uns aos outros e a aplicar o que eles aprenderam em suas próprias pesquisas científicas.

Público alvo: Este curso se destina especificamente a alunos de mestrado em oceanografia.

Importante: Os alunos devem trazer um laptop (mínimo de um por dupla, idealmente) já preparado com os softwares que serão utilizados na aulas. Os alunos são fortemente encorajados a ler o Code of Conduct do Software Carpentry's. É um projeto open source muito bacana no qual me baseei para estruturar este curso. Nesta ocasião, devido ao modo remoto como o curso será conduzido, o curso está condensado em cerca de metade do conteúdo proposto por este grupo. A leitura das Lessons propostas por eles é muito interessante como preparação para as aulas, principalmente daquelas que trataremos nas aulas (Python, Git).
